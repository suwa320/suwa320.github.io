<a href="https://farm1.staticflickr.com/830/27199163467_60144c42da_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 1日目 - 通宵目指して峠を登る" src="https://farm1.staticflickr.com/830/27199163467_66f1ccc15c_c.jpg"></a>
去年の続きの徒歩旅行。去年はニャロンからガンゼ(甘孜)に向かうつもりだったが、地図を見ているとニャロンからアチェンガル・ゴンパ(亜青寺)まで車道が通っている。

体慣らしにしばらくは車道を歩こうと思っていたので、なんとなく今年もガンゼかなと思っていたが車道が通っているならアチェンガル・ゴンパに寄ってみるかという気になった。アチェンガル・ゴンパはラルンガル・ゴンパと並ぶ東チベットの目玉観光地なので見ておいて損はないはずだ。

朝7時半に宿を出る。食堂で朝食を食べようと思っていたが食欲がまだないので朝食を食べずに歩き始める。

<br>



<a href="https://farm1.staticflickr.com/943/40261684160_6de069801e_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 1日目 - 通宵目指して峠を登る, 1" src="https://farm1.staticflickr.com/943/40261684160_7809be14fd_c.jpg"></a>
ニャロンからアチェンガル・ゴンパに向かうには、州道を南に25キロほど下りそこから郷道を西に進みピッツァに行き、ピッツァから北上して通宵に行く、 通宵から西に進みアチェンガル・ゴンパに行く。

しかしピッツァは去年、リタン(理塘)からニャロンに向かう時に通ったので、ニャロンからアチェンガル・ゴンパに向かうのに2日間ほど去年歩いた道を歩かないといけない。

地図を見ているとニャロンから州道を10キロ程度南に下り、そこから山を越えて通宵に出れるようだ。出発してすぐに山越えはハードだが同じ道を歩くよりはいい。通宵まで出ればあとはアチェンガル・ゴンパまでは車道ので大丈夫だろう。

<br>


<a href="https://farm1.staticflickr.com/826/41169406045_d8de93215f_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 1日目 - 通宵目指して峠を登る, 2" src="https://farm1.staticflickr.com/826/41169406045_0ef95642df_c.jpg"></a>
州道を南に下って行き、12時前に通宵に向かう道との分岐に着く。

<br>



<a href="https://farm1.staticflickr.com/955/27199668837_e337ceec6c_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 1日目 - 通宵目指して峠を登る, 3" src="https://farm1.staticflickr.com/955/27199668837_92760bb689_c.jpg"></a>
道はまだ緩やかで集落の中を通って行く。


<br>

<a href="https://farm1.staticflickr.com/944/28196046168_33793e831c_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 1日目 - 通宵目指して峠を登る, 4" src="https://farm1.staticflickr.com/944/28196046168_4a9d25e0ec_c.jpg"></a>
集落を過ぎると急斜面を登って行く。


<br>

<a href="https://farm1.staticflickr.com/943/41348686034_7cb6feae2e_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 1日目 - 通宵目指して峠を登る, 5" src="https://farm1.staticflickr.com/943/41348686034_7caf2e07ba_c.jpg"></a>
急斜面を登りきっても緩やかな登りが続く。道が分岐していて、どちらに進んでいいのか分からないので、人が来まで待つ。

バイクに乗ったチベット人が来たので、どちらの道に進んだらいいのか聞く。ついでにどこかに水場がないかと聞くと、あそこにあるよというので見てみると100メートルほど先にタルチョが道の脇になびいているところがある。そこが水場のようだ。

水場も近くにあるし今日はこの辺でテント泊することにした。

<br>

<a href="https://farm1.staticflickr.com/912/27198472027_ce427ef276_o.jpg"><img alt="ニャロン(新龍)からアチェンガル(亜青) 1日目 - 通宵目指して峠を登る, 6" src="https://farm1.staticflickr.com/912/27198472027_2125283f5c_c.jpg"></a>
夜は成都で買っておいたカレーを作って食べた。
