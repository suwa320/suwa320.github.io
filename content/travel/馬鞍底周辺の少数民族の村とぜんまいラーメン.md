 <a href="https://farm1.staticflickr.com/953/41170042525_d7472caba3_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン" src="https://farm1.staticflickr.com/953/41170042525_8cd1c4978e_c.jpg"></a>
朝から霧が出ている。部屋にいても寒いので9時前に米線でも食べに出る。外に出ると霧雨が降っている。

食堂に入り米線を頼む(6元、120円)。米線を食べていると日本語が聞こえてくる。店のおじさんがタブレットで日本のアニメを見ているようだ。おじさんまで日本のアニメを見ているとは日本のアニメや漫画は随分と普及しているものらしい。

<br>

<a href="https://farm1.staticflickr.com/908/42024020802_55fe639d94_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 1" src="https://farm1.staticflickr.com/908/42024020802_825b239954_c.jpg"></a>
米線を食べて外に出ると霧雨もやんでいたので少し歩いてみることにする。馬鞍底中学からそのまま先に歩いて行く。しばらく歩くとハニ族の村があった。

<br>

<a href="https://farm1.staticflickr.com/953/40261890720_740e2887f2_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 2" src="https://farm1.staticflickr.com/953/40261890720_f42cf955df_c.jpg"></a>
村には建設途中のレンガを積んだ家がたくさんあった。建てるのをやめてしまったのか、ほったらかしにされているような家も多かった。村の規模から考えたら異常に大きな建てかけの家もあった。

<br>

<a href="https://farm1.staticflickr.com/904/27199822647_47fa1f606f_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 3" src="https://farm1.staticflickr.com/904/27199822647_a11f8439a0_c.jpg"></a>
村から別の村に道が出ていないかと探したが見つけられなかった。山に入って行く山道があったので登ってみる。

<br>

<a href="https://farm1.staticflickr.com/958/41348794304_925e7557b9_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 4" src="https://farm1.staticflickr.com/958/41348794304_72dde2f332_c.jpg"></a>
急な登りを1時間ちょっと登ると山道が草や枝に覆われてくる。仕方ないので引き返して途中にあった分かれ道を別の方に進んでみるが、またしばらくすると草や枝に覆われてくる。頑張って進んでも何もなさそうなので村に戻る。

<br>

<a href="https://farm1.staticflickr.com/912/41170076135_35edb0261c_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 5" src="https://farm1.staticflickr.com/912/41170076135_bd6e47b9c4_c.jpg"></a>
村に入る手前に分かれ道があったのでその道を歩いてみる。最近、造られたか補強された道のようだ。

<br>

<a href="https://farm1.staticflickr.com/907/42068728131_c7ec96f14f_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 6" src="https://farm1.staticflickr.com/907/42068728131_5b67b9f65a_c.jpg"></a>
ハニ族の集落があり棚田が広がっている。下には公道が走っている。馬鞍底から地西北に行く道だろう。

<br>

<a href="https://farm1.staticflickr.com/907/27199812537_53a4993877_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 7" src="https://farm1.staticflickr.com/907/27199812537_1f18032b1b_c.jpg"></a>
棚田の中に狛犬のようなものが置かれていた。

<br>

<a href="https://farm1.staticflickr.com/827/40260969310_0719f05d39_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 8" src="https://farm1.staticflickr.com/827/40260969310_febe043dec_c.jpg"></a>
公道をもっと下がっていくときれない棚田が見れそうだったが、霧が晴れてきたので宿で洗濯をしようと思い戻ることにする。

1時半頃、宿に戻ると宿のおじさんがいたので、少し話すとここから3キロぐらい歩いたところにハニ族とヤオ族の村があると教えてくれた。洗濯をして、少し休憩をして、また歩きに出かける。

<br>

<a href="https://farm1.staticflickr.com/906/41169056605_4566f9ed59_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 9" src="https://farm1.staticflickr.com/906/41169056605_3e6357fbf0_c.jpg"></a>
雲が取れてすっかり天気がよくなっている。遠くの山までよく見える。

<br>

<a href="https://farm1.staticflickr.com/958/27199315967_cdc78e629b_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 10" src="https://farm1.staticflickr.com/958/27199315967_4771e75d65_c.jpg"></a>
山の上にある馬鞍底の町がよく見える。

<br>

<a href="https://farm1.staticflickr.com/981/27199093097_2d97c123c8_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 11" src="https://farm1.staticflickr.com/981/27199093097_96f62d0c4e_c.jpg"></a>
細い車道を登りハニ族の村へ行く。

<br>

<a href="https://farm1.staticflickr.com/957/41349745624_36baa15538_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 12" src="https://farm1.staticflickr.com/957/41349745624_dde0c274a5_c.jpg"></a>
刺繍をしている人が多い。

村の先にまだいくつか村があるが、日が暮れそうなので帰ることにする。

<br>

<a href="https://farm1.staticflickr.com/954/41169054055_653f517fdb_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 13" src="https://farm1.staticflickr.com/954/41169054055_a1183496e9_c.jpg"></a>
午前中歩いた時、村にたどり着けない山道を随分登ってしまったが、ぜんまいがたくさん生えていたので摘んでおいたのだ。宿に戻り、焚き火の跡から持ってきた灰であく抜きをする。

<br>

<a href="https://farm1.staticflickr.com/976/41350166594_30193d9b87_o.jpg"><img alt="馬鞍底周辺の少数民族の村とぜんまいラーメン, 14" src="https://farm1.staticflickr.com/976/41350166594_8c5e8bdf28_c.jpg"></a>
数日後、インスタントラーメンに載せて食べた。少し苦味が残っていたがおいしかった。

<br>
